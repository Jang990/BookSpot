# 책 검색 기능

- 책 검색 API
  - 책 제목 검색 - title
  - Slice로 반환
    - 10권 단위 페이징

- 책 검색 UI
  - 책 검색바
  - 책 검색 결과
    - 무한 스크롤

## 나중에 해야할 일
1. 예외처리 필요
   1. API - 페이지 크기 제한
   2. API - 검색어 2글자 이상으로 제한
   3. UI - 검색결과
2. UI - 스크롤을 내려서 추가 검색중일 때 로딩을 사용자가 알아챌 수 없음. 개선 필요
3. 책 제목 이외의 검색 기능이 필요함. ex) 출판사
4. 책의 수가 많을 때 어떤 방식으로 정렬해야 할까?
   1. 최신순?
   2. 사용자가 선택한 횟수가 많은 순?
